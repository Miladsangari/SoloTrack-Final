# SoloTrack Backend

Node + Express + MongoDB API for the SoloTrack Incident Management project.

## Quick start

1. Go to the backend folder:

   ```bash
   cd backend
   npm install
   ```

2. Create a `.env` file based on `.env.example`.

3. Run the API:

   ```bash
   npm run dev
   ```

API will run on `http://localhost:3000`.
