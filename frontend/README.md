# SoloTrack Frontend

React frontend for the SoloTrack Incident Management project.

## Quick start

1. Go to the frontend folder:

   ```bash
   cd frontend
   npm install
   ```

2. Optional: create a `.env` file to override the API base URL:

   ```env
   VITE_API_BASE=http://localhost:3000
   ```

3. Run the dev server:

   ```bash
   npm run dev
   ```

App will run on `http://localhost:5173`.
